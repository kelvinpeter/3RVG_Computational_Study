\# Computational Study of 3RVG with Compound F5857\_2965



\## Repository Structure

\- `1\_DOCKING/` - Molecular docking files and results

\- `2\_MD\_SIMULATION/` - Molecular dynamics setup and outputs

\- `3\_TRANSCRIPTOMICS\_ANALYSIS/` - Differential expression analysis of GSE16161 psoriasis dataset



\## Quick Access

\- \*\*Top docking pose\*\*: `1\_DOCKING/IFD\_DRug\_F5857\_\_2965\_TOP\_IDENTIFIED\_3RVG.pdb`

\- \*\*MD setup\*\*: `2\_MD\_SIMULATION/desmond\_setup\_F5857\_3RVG-out.cms`

\- \*\*DEG analysis script\*\*: `3\_TRANSCRIPTOMICS\_ANALYSIS/GSE16161\_DEG\_analysis.R`



\## Software Requirements

\- Schrödinger Suite for docking files

\- Desmond for MD files

\- R 4.2.2+ with GEOquery, limma, umap for transcriptomics
